Um portal projetado para oferecer informações climáticas atuais.

---

## 🔧 Instalação
Use o seguinte comando para instalar as dependências:

```bash
npm install
```
Use o seguinte comando para rodar o projeto em tempo real na sua máquina:

```bash
npm run dev
```
